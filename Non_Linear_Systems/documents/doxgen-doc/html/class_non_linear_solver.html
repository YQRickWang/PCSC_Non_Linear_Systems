Newton method is a root-finding algorithm which produces successively better approximations to the roots (or zeroes) of a real-valued function Given the function f and the initial guess X0, the iteration goes like Xn+1 = Xn - f(Xn)/fp(Xn), where fp is the derivative of the function f.

Parameters
initial_guessthe initial guess of the solution
max_iterationsthe limit of the number of iterations of the method
